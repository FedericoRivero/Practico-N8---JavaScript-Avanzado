Explica con tus propias palabras los siguientes conceptos.

- Que es un hashTable?
    Es una estructura de datos asociativa que permite almacenar datos de forma que se pueda acceder a ellos de forma directa a partir de una clave que identifica a cada elemento de la tabla, esto lo hace muy rapido y eficiente.

- Que hace un get
    Obtiene el valor asociado a la clave que se le pasa como parametro.

- Que hace un set
    Asocia un valor a una clave que se le pasa como parametro.
